# 竞品分析指南

## 竞品识别

### 竞品类型

**直接竞品**
- 提供相同产品/服务
- 目标相同客户群
- 直接争夺市场份额

**间接竞品**
- 解决相同问题但方式不同
- 可能是替代方案或互补方案

**潜在竞品**
- 可能进入该市场的玩家
- 大厂的相关产品线

### 竞品发现渠道

通过搜索发现：
- "[产品类型] 厂商"
- "[产品类型] 排名"
- "[产品类型] 市场格局"
- "[竞品名] 替代品"
- "[竞品名] vs"

## 竞品内容策略分析

### 分析维度

**内容定位**
- 目标受众是谁？
- 内容调性如何？（技术向/商业向）
- 差异化定位是什么？

**内容主题**
- 主要内容支柱有哪些？
- 高频话题是什么？
- 是否有独特的内容角度？

**内容形式**
- 博客文章
- 白皮书/电子书
- 案例研究
- 视频/播客
- 网络研讨会
- 文档/教程

**内容质量**
- 深度如何？
- 是否有原创洞察？
- 是否有数据支撑？

**SEO 表现**
- 排名靠前的关键词有哪些？
- 内容更新频率？
- 外链建设情况？

### 分析模板

对每个竞品收集以下信息：

```
竞品名称：
官网 URL：
核心定位：
目标客户：
主要内容主题：
  - 主题1
  - 主题2
  - 主题3
内容形式：
内容更新频率：
特色/亮点：
不足/机会点：
```

## 差异化机会识别

### 内容差距分析

找出竞品未覆盖或覆盖不足的领域：

**话题差距**
- 有哪些话题竞品没有深入讨论？
- 有哪些新兴话题还没人写？

**深度差距**
- 哪些话题只有浅层内容？
- 哪些复杂问题缺乏详细解答？

**形式差距**
- 是否缺少某类内容形式？
- 是否有内容本地化机会？

**受众差距**
- 是否有被忽视的细分受众？
- 是否有特定行业/角色未被服务好？

### 差异化策略

**深度差异化**
- 在相同话题上提供更深入、更全面的内容

**角度差异化**
- 从独特视角解读相同话题

**形式差异化**
- 用不同形式呈现内容（如：把文章做成交互式工具）

**受众差异化**
- 针对特定细分市场定制内容

**时效差异化**
- 更快响应行业动态和趋势
