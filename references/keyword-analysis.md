# 关键词分析指南

## B2B 关键词特点

B2B 关键词与 B2C 有显著差异：

| 维度 | B2B | B2C |
|-----|-----|-----|
| 搜索量 | 较低 | 较高 |
| 转化价值 | 高（大客单价） | 低（小客单价） |
| 决策周期 | 长（周/月） | 短（分钟/天） |
| 搜索者 | 专业人士 | 普通消费者 |
| 关键词类型 | 专业术语多 | 口语化多 |

## 搜索意图分类详解

### 信息型 (Informational)
用户目的：学习、了解、研究

特征词：
- 什么是、如何、为什么、怎样
- 教程、指南、入门、介绍
- 原理、概念、定义

内容匹配：
- 知识库文章
- 入门指南
- 概念解读
- 教程系列

### 导航型 (Navigational)
用户目的：找到特定网站或页面

特征词：
- 品牌名 + 官网/登录/下载
- 产品名 + 文档/API

内容匹配：
- 官网优化
- 产品页面
- 文档中心

### 商业型 (Commercial Investigation)
用户目的：评估、比较、准备购买

特征词：
- vs、对比、比较、哪个好
- 排名、推荐、最佳、TOP
- 评测、评价、选型

内容匹配：
- 对比文章
- 选型指南
- 产品评测
- 排行榜

### 交易型 (Transactional)
用户目的：准备采取行动（购买、注册、下载）

特征词：
- 价格、报价、费用、多少钱
- 试用、免费、Demo、申请
- 购买、订阅、开通

内容匹配：
- 定价页面
- 免费试用页
- 产品演示

## 关键词优先级评估

### 评估维度

**相关性 (1-5)**
- 5：直接描述产品/服务
- 4：描述产品类别
- 3：描述相关场景/痛点
- 2：间接相关
- 1：边缘相关

**商业价值 (1-5)**
- 5：交易型意图明确
- 4：商业调研意图
- 3：混合意图
- 2：信息型为主
- 1：纯信息型

**竞争难度 (1-5)**
- 5：竞争极低，容易排名
- 4：竞争较低
- 3：竞争适中
- 2：竞争较高
- 1：竞争激烈，难以排名

**优先级计算**
综合得分 = 相关性 × 0.4 + 商业价值 × 0.35 + 竞争难度 × 0.25

- 高优先级：≥ 3.5
- 中优先级：2.5 - 3.4
- 低优先级：< 2.5

## 长尾关键词策略

### 长尾关键词优势
- 竞争度低
- 意图更明确
- 转化率更高
- 适合新站点起步

### 长尾关键词挖掘方法

**问题式**
- 如何解决 [问题]
- [场景] 怎么办
- 为什么 [现象]

**对比式**
- [产品A] 和 [产品B] 哪个好
- [产品] vs [竞品]
- [产品] 替代方案

**场景式**
- [行业] + [产品类型]
- [规模] 企业 + [需求]
- [角色] 用什么 [工具]

**修饰式**
- 最佳/最好的 [产品类型]
- [年份] [产品类型] 推荐
- 免费/开源 [产品类型]

## 关键词与内容映射

每个关键词应对应一篇主要内容：

```
关键词 → 内容类型 → 买家阶段 → 内容形式
```

示例：
```
"什么是数据中台" → 概念解读 → 认知阶段 → 博客文章
"数据中台 vs 数据仓库" → 对比分析 → 考虑阶段 → 博客文章
"数据中台选型指南" → 选型指导 → 考虑阶段 → 白皮书
"数据中台实施案例" → 案例分析 → 决策阶段 → 案例研究
```
